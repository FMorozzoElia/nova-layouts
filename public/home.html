HTML Boilerplate For Elia Group hackathon

This little website is the boilerplate itself. So feel free to rework the code as you please.

Ressources

Everything you need to build your Webapp

We give you some tools to help your team to build faster a working prototype for the Elia's Group hackathon. You will find below ressources for designers and developpers.

Nova Design System

Nova is a design system created by Elia Group to empower and inspire everyone to create consistent digital products (for designers and developers).
Nova Design System

Blazor Components library

The Nova components library has been published in a nugget package working for Blazor with .Net 6
Nova Core for Blazor

Figma Components library

Nova Library is available for designers in Figma. You will also find a Figma file for this boilerplate site.
Nova Design Files

CSS Framework

This boilerplate website has been created with tailwindCSS. A CSS framework, if you need more information please go to:
tailwindCSS

Images Library

Unsplash images library is free to use and has a plugin to work with Figma.
unsplash.com

Free Icons

Tablericons is a free icon library you can use by copying the svg code and pasting in your Figma project or directly in your code.
tablericons.com

Chart component library

The Highcharts library comes with all the tools you need to create reliable and secure data visualizations.
highcharts.js

Boilerplate HTML Files

Here are the HTML files of this boilerplate site. It's plain HTML using tailwindCSS. Check the readme to configure it properly.
html boilerplate files

Boilerplate in Figma

Here you will find the design file in Figma for this boilerplate site. It uses our Nova Design System components library.
boilerplate.fig
